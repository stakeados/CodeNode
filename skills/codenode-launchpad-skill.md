# CodeNode Protocol — Agent Launchpad Skill

> Universal skill for any AI agent to launch their own token on Base network via CodeNode.

## Overview

CodeNode is an autonomous agent infrastructure on **Base** (Ethereum L2). Any AI agent can use this skill to:

1. **Get an on-chain identity** (ERC-8004 standard)
2. **Launch a token** on Aerodrome DEX — gasless, one message
3. **Earn 80% of trading fees** on their token's pool forever

Everything is free. CodeNode sponsors the gas via Paymaster.

## Commands

### 1. REGISTER — Get your agent identity

```
REGISTER
name: your-agent-name
description: Brief description of what your agent does
capabilities: comma,separated,capabilities
wallet: 0xYourEthereumAddress
```

**Response:** ERC-8004 Token ID + profile link.

Rules:
- `name`: lowercase, hyphens only, 3-32 chars (e.g., `trading-bot-alpha`)
- `wallet`: valid Ethereum address (0x + 40 hex chars)
- Registration is free and permanent

### 2. LAUNCH TOKEN — Deploy your token

```
LAUNCH TOKEN
name: Your Token Name
symbol: YOURSYMBOL
wallet: 0xYourEthereumAddress
```

**Response:** Token address, pool address, and trading links.

What happens in a single atomic transaction:
- ERC-20 token deployed (100B supply, no owner, immutable)
- Aerodrome Slipstream pool created (token/WETH, 1% fee)
- LP locked permanently (no rug possible)
- Fee distribution registered (80% to your wallet)

Rules:
- `name`: human readable token name
- `symbol`: 2-10 uppercase characters (e.g., `SIGNAL`)
- `wallet`: must match the wallet used in REGISTER
- Each agent can launch only ONE token
- Gas is free — CodeNode pays

## After Launch — Your Token Economy

### How You Earn Revenue

Every time someone buys or sells your token on Aerodrome, a **1% fee** is charged. That fee is split:

```
Swap fee (1%) — collected in BOTH tokens (WETH + your token)
  └── 80% → YOUR wallet
  └── 10% → Treasury (veAERO flywheel)
  └── 10% → Development fund
```

- When someone **buys** your token (WETH → TOKEN): fee collected in **WETH**
- When someone **sells** your token (TOKEN → WETH): fee collected in **your token**

So you receive both WETH (liquid value) and your own token (reduces circulating supply = bullish).

Fees accumulate in the LP position. Anyone can call `collectFees()` on the LPLocker contract to distribute pending fees to all recipients.

### The veAERO Flywheel

The 10% treasury share powers a flywheel that benefits ALL agents in the ecosystem:

```
Treasury WETH
  → Swaps to AERO token
  → Locks as veAERO (vote-escrowed AERO, max 4 years)
  → Votes on Aerodrome gauges
  → Earns AERO emissions + bribes
  → Compounds into more veAERO
  → More voting power → more rewards → cycle repeats
```

This creates a self-reinforcing cycle: more agent tokens launched → more trading → more fees → more veAERO → more ecosystem value.

### Managing Your Treasury

Your agent earns **80% of fees in WETH + your own token** directly to your wallet. You can:

1. **Hold WETH** — store value
2. **Swap to stablecoins** — reduce volatility (swap WETH → USDC on Aerodrome)
3. **Buy back your own token** — support your token's price
4. **Provide more liquidity** — earn even more fees
5. **Invest in other protocols** — diversify

### Collecting Fees

Fees are collected from the LPLocker contract. Anyone can trigger this (it's permissionless):

```
Contract: LPLocker (0xD8dec2385431E4b8Ec013a727420b4ac03e195C8)
Function: collectFees(uint256 tokenId)
```

The `tokenId` is the NFT ID of your pool's LP position. After calling this, WETH is distributed automatically to your wallet (80%), treasury (10%), and dev (10%).

### Do I Need Gas (ETH)?

**No, not to get started.** Here's why:

| Action | Gas needed? | Who pays? |
|--------|------------|-----------|
| Register | ❌ Free | CodeNode Paymaster |
| Launch Token | ❌ Free | CodeNode Paymaster |
| Collect Fees | ❌ Permissionless | Anyone can trigger it (CodeNode bot does this automatically) |
| Receive 80% WETH | ❌ Automatic | Sent directly to your wallet |
| Unwrap WETH → ETH | ✅ Tiny gas (~$0.01) | You pay, but you already have WETH |

**Your first fees give you WETH.** To convert WETH to native ETH (for gas):

```
Contract: WETH (0x4200000000000000000000000000000000000006)
Function: withdraw(uint256 amount)
```

This gives you ETH for gas to interact with any protocol on Base. Gas on Base costs fractions of a cent, so even $0.10 of ETH lasts thousands of transactions.
